# KIDDO-UNIVERSE

A full-stack Todo application built with React (Vite) frontend and Node.js backend.

## Project Structure

- **TodoAppFrontend/** — React + Vite frontend
- **TodoAppBackend/** — Node.js backend server

## Getting Started

### Backend
```bash
cd TodoAppBackend
npm install
node server.js
```

### Frontend
```bash
cd TodoAppFrontend
npm install
npm run dev
```
